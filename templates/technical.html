Definition

C is a high-level and general-purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System in the early 1970s.

Hello World

Here is an example of a C program:

int main(void)

{

printf("hello, world\n");

}

Conditions

Conditions in C would look something like this:

if (x < y)

{

printf("x is less than y\n");

}

else if (x > y)

{

printf("x is greater than y\n");

}

else if (x == y)

{

printf("x is equal to y\n");

}

Loops

Loops can be written like the following:

while (true)

{

printf("hello, world\n");

}

The above loop is called a while loop, but there is also different kinds of loops, like for loops:

for (int i = 0; i < 50; i++)

{

printf("hello, world\n");

}

Variables

Different variables can also be declared in the following way:

  1. bool x;
  2. char char;
  3. double num;
  4. float f;
  5. int i;
  6. string s;

Arrays

In memory, we can store variables one after another, back-to-back. And in C, a list of variables stored, one after another in a contiguous chunk of memory, is called an array. Strings are actually just arrays of characters. We can see this below:

int main(void)

{

string s = get_string("Input: ");

printf("Output: ");

for (int i = 0; i < strlen(s); i++)

{

printf("%c\n", s[i]);

}

}
